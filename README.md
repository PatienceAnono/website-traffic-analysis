# 📊 Website Traffic & Revenue Analysis

## 📌 Overview
This project analyzes website traffic, marketing performance and user behavior to identify key drivers of revenue and optimization opportunities.

---

## 🎯 Objectives
- Identify top-performing traffic sources
- Evaluate campaign effectiveness
- Analyze user behavior across devices
- Discover revenue trends

---

## 🛠 Tools Used
- Python (Data Cleaning & Analysis)
- Power BI (Dashboard & Visualization)

---

## 📊 Key Insights
- Email Marketing generated the highest revenue
- Direct and Organic Search had the highest conversion rates
- Mobile drove the most traffic but had lower conversion efficiency
- Revenue peaked during March and weekends
- Conversion rate drives revenue more than traffic volume
